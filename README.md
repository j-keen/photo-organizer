# 📸 사진영상정리기 (Photo Organizer)

브라우저에서 실행되는 로컬 사진/영상 정리 웹앱입니다.

서버 없이 HTML 파일 하나로 동작하며, 모든 데이터는 로컬에 저장됩니다.

## ✨ 주요 기능

### 📁 파일 관리
- 폴더 선택 후 사진/영상 갤러리 표시
- 드래그 앤 드롭으로 파일 이동
- 폴더 생성/삭제
- 휴지통 기능

### 🏷️ 태그 & 메모
- 파일별 메모 저장 (Markdown 지원)
- 태그 칩 UI + 자동완성
- 사이드카 파일(.txt)로 메타데이터 저장

### ⭐ 정리 기능
- 별점 평가 (1-5점)
- 드래그로 다중 선택 (Lasso)
- 일괄 이름 변경
- 정렬 옵션 (이름, 날짜, 크기, 평점)

### 🎬 비디오 기능
- 타임라인 마커 추가
- 스크린샷 캡처
- 키보드 단축키 (J/K/L, 화살표)
- 이동 간격 설정 (1-60초)

## 🚀 사용법

1. `사진영상정리기.html` 파일을 브라우저에서 열기
2. "폴더 열기" 버튼 클릭
3. 정리할 사진/영상이 있는 폴더 선택
4. 끝!

## ⌨️ 단축키

| 키 | 기능 |
|---|---|
| `←` `→` | 영상 앞/뒤로 이동 |
| `J` / `L` | 뒤로/앞으로 건너뛰기 |
| `K` / `Space` | 재생/일시정지 |
| `M` | 마커 추가 |
| `C` | 빠른 캡처 |
| `Ctrl+←` `Ctrl+→` | 이전/다음 파일 |

## 🔧 기술 스택

- **Vanilla JavaScript** - 프레임워크 없음
- **File System Access API** - 로컬 파일 접근
- **IndexedDB** - 설정/캐시 저장

## 📋 지원 포맷

- **이미지**: JPG, PNG, GIF, WebP, BMP
- **영상**: MP4, WebM, MOV
